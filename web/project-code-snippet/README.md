## 如何实现一个图片懒加载
> 懒加载（Lazy loading），也被称为延迟加载，通过延迟加载网页中的图片，只在图片进入可视区域时才进行加载，从而提升网页加载性能和用户体验。

原理是 只加载当前屏幕的图片，可视区域外的图片不会进行加载，只有当屏幕滚动的时候才加载。
